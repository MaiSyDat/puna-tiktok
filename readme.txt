A modern WordPress theme that creates a TikTok-like video sharing platform with sidebar navigation and video streaming capabilities.

== Description ==

TikTok Clone is a professional WordPress theme designed to create a video sharing platform similar to TikTok. It features a modern sidebar navigation, video grid layout, and interactive video player with like, comment, and share functionality.
