A modern WordPress theme that creates a TikTok-like video sharing platform with sidebar navigation and video streaming capabilities.

== Description ==

Puna TikTok is a professional WordPress theme designed to create a video sharing platform similar to TikTok. It features a modern sidebar navigation, video grid layout, and interactive video player with like, comment, share, and save functionality.

**Key Features:**

* **Multiple Video Sources:**
  * Upload videos directly to your server (MP4, WebM, OGG, MOV, AVI)
  * Embed YouTube videos (supports both regular links and Shorts)
* **Interactive Features:**
  * Like, comment, share, and save videos
  * Reply to comments with nested replies
  * Real-time interaction updates
* **Modern UI/UX:**
  * TikTok-inspired vertical video feed
  * Smooth scrolling between videos
  * Sidebar navigation with active page highlighting
  * Search functionality with history
* **Responsive Design:**
  * Optimized for desktop, tablet, and mobile devices
  * Touch-friendly controls
* **Customization:**
  * Light/Dark mode (auto-detect system preference)
  * Customizable colors via WordPress Customizer
  * Custom logo and footer settings
* **Performance:**
  * Lazy loading for videos and images
  * Optimized asset loading
  * Minimal database queries

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through the 'Appearance' menu in WordPress
3. The theme will automatically create necessary pages (Explore, Followed, etc.)
4. Go to Customize to configure colors, logo, and footer

== Frequently Asked Questions ==

= What video formats are supported? =
The theme supports MP4, WebM, OGG, MOV, and AVI video formats for local uploads, plus YouTube video embeds.

= Can I change the theme colors? =
Yes! Go to Appearance → Customize → Colors to customize the theme colors.

= Does it support mobile devices? =
Yes, the theme is fully responsive and optimized for mobile devices.

= Can I use YouTube videos? =
Yes, you can embed YouTube videos by pasting YouTube links (regular or Shorts) in the video upload section.
